# Documentation Images

This directory contains images used throughout the Flipnzee Analytics GitHub repository, including the README, project documentation, GitHub Social Preview, architecture diagrams, screenshots, and promotional materials.

These assets are maintained separately from the WordPress plugin to keep documentation resources independent of the plugin source code.

## Directory Structure

```text
docs/
└── images/
    ├── flipnzee-analytics-github-banner.png
    ├── flipnzee-analytics-marketing-banner.png
    ├── flipnzee-analytics-social-preview.png
    ├── architecture-diagram.png
    ├── dashboard-overview.png
    ├── search-console.png
    ├── countries.png
    └── frontend-widget.png
```

### Description

| File | Purpose |
|------|---------|
| flipnzee-analytics-github-banner.png | Banner displayed at the top of the GitHub README |
| flipnzee-analytics-marketing-banner.png | Promotional artwork for articles and documentation |
| flipnzee-analytics-social-preview.png | GitHub Social Preview image shown when the repository is shared |
| architecture-diagram.png | High-level architecture of the plugin |
| dashboard-overview.png | Analytics dashboard screenshot |
| search-console.png | Google Search Console integration screenshot |
| countries.png | Geographic visitor insights screenshot |
| frontend-widget.png | Public frontend analytics widget screenshot |
